# Trabalho-AED
  Um repositório criado para o desenvolvimento de um trabalho da matéria de Algoritmos e Estruturas de Dados.
  Este algoritmo implementa a estrutura fila para o controle de um estácionamento fictício, usando o método de organização FIFO (First In First Out) o algoritmo simula um sistema de gerência que permite estacionar até 5       veículos com suas respectivas placas e marcas, a aplicação tambem permite a remoção de veículos por seus índices
